# 👤 Profile Card

This is a simple and responsive **Profile Card** built using HTML and CSS. It showcases a user’s photo, name, title, and contact button along with social media icons.

## 📌 Features

- Responsive design
- Profile picture with rounded styling
- Name and profession
- Social media icons
- "Contact Me" button
- Clean and modern UI
